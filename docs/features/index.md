# Feature Guides

## Latest Features (Q4 2025)

- **[Human-in-the-Loop (HITL)](hitl.md)** – Pause AI tool execution for user approval before risky operations
- **[Guardrails Middleware](guardrails.md)** – Content filtering, PII detection, and safety checks for AI outputs
- **[Redis Conversation Export](conversation-history.md)** – Export session history as JSON for analytics and debugging

## Core Features (Q3 2025)

- **[Multimodal Chat Experiences](multimodal-chat.md)** – Stream text + images with automatic provider fallbacks
- **[Auto Evaluation Engine](auto-evaluation.md)** – Automated quality scoring and metrics export
- **[CLI Loop Sessions](cli-loop-sessions.md)** – Persistent interactive mode with conversation memory
- **[Regional Streaming Controls](regional-streaming.md)** – Region-specific model deployment and routing
- **[Provider Orchestration Brain](provider-orchestration.md)** – Adaptive provider/model selection with fallbacks
