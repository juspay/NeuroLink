# CLI Screenshots

This directory is designated for professional CLI screenshots demonstrating NeuroLink CLI capabilities.

## Planned Screenshots

The following screenshots need to be captured:

1. **01-cli-help** - Complete CLI help command output showing all available commands
2. **02-provider-status** - Provider connectivity status with all providers verified
3. **03-text-generation** - Real AI text/haiku generation with JSON output
4. **04-best-provider** - Automatic provider selection demonstration
5. **05-batch-results** - Multi-prompt batch processing with results

## Capture Guidelines

- Resolution: 1920x1080 minimum
- Terminal theme: Professional dark theme preferred
- Include actual AI responses (not mocked data)
- Use timestamped filenames: `[##]-[name]-YYYY-MM-DDTHH-MM-SS.png`

## Recording Tools

Recommended tools for CLI screenshots:
- **macOS**: Built-in screenshot (Cmd+Shift+4) or iTerm2 screenshot feature
- **Linux**: GNOME Screenshot, Flameshot, or Shutter
- **Windows**: Windows Terminal screenshot or Snipping Tool

For asciinema recordings (convertible to GIF), see `docs/cli-recordings/`.
